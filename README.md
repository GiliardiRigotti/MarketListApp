## 🚀 How to build and run locally

- [Setup development Environment](https://reactnative.dev/docs/environment-setup)
- Build/Run on iOS 🍎
```
yarn ios
```
```
npm run ios
```
- Build/Run on Android 🤖
```
yarn android
```
```
npm run android
```
## 💻 Start the Dev Client

```
expo start --dev-client
```
```
yarn start
```
```
npm run start
```
